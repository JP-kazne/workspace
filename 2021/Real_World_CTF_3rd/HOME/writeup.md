# Writeup

HOME画面(https://realworldctf.com/)のソースコード等々を調べてみると、以下の画像ファイルを発見。

![](img/signin.baacf08.jpg)

画像の内容に従って接続し、データを受信する。

```py
from pwn import *
io = remote('home.realworldctf.com',1337)
msg = io.recvall()
print(msg.decode('ascii'))
io.close()
```
